# Sistema de Conciliação de Dados Financeiros

Projeto desenvolvido em Python com o objetivo de simular um processo de conciliação de dados financeiros entre sistemas distintos.

## Funcionalidades:
- Leitura de arquivos CSV
- Comparação de dados entre sistemas
- Identificação de divergências
- Geração automática de relatório


## Stack Tecnológica:
- Python
- Pandas

## Objetivo:
Simular um processo real de validação e conciliação de dados, garantindo precisão e confiabilidade das informações.

## Como executar:
1. Faça upload dos arquivos sistema.csv e corretora.csv
2. Execute o script conciliacao_dados.py
3. O sistema irá gerar um relatório com divergências encontradas
